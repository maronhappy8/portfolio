WORK

Shisha lounge Persia

  • TYPE: Web site
  • URL: http://persia.m-miri.jp
Scroll

福岡で唯一のステンドグラス風シーシャを取り扱っている「Shisha lounge Persia」さまのWebサイト新規立ち上げを担当させていただきました。シーシャの危険なイメージを払拭したいというクライアントの思いのもと、初心者でも入店しやすい情報設計や店内の雰囲気が伝わるサイトデザインを目指しました。

  • ターゲット

    普段からシーシャを嗜んでいる、シーシャに興味はあるが未経験の２０代男女

  • 目的

    シーシャの正しいイメージを伝える、落ち着いた空間でリラックスしてもらう、店舗に足を運ぶことのハードルを下げてもらう

  • 情報設計

    福岡で唯一のステンドグラス風ボトルを取り扱っていることがアピールできる構成やライティングを考えました。また、初心者の方が入店しやすいようにビギナーズガイドを用意することで、シーシャの正しいイメージを伝えらるよう心がけました。
    FAQではユーザーの潜在的な疑問を挙げることで多様な楽しみ方を発信しています。

  • デザイン

    店舗全体の雰囲気を感じることができるよう、店舗の外観である石畳みやグレーのレンガをイメージしてデザインしました。また店内の照明を意識し、温かみのあるグレーも使用しています。
    余白を多くとり、ゴールドのあしらいを置くことで落ち着いた空間や高級感を演出しています。
    また、リッチなアニメーションで単調すぎないイメージに仕上げました。

  • 制作範囲・期間

    企画、ワイヤーフレーム : 1週間
    デザイン、ライティング : 2週間
    コーディング : 1ヶ月

  • 使用ツール

    XD / Illustrator / Photoshop

  • 使用言語

    HTML / Sass / jQuery / Javascript
